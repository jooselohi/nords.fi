Ilmainen hosting-palvelu Google Firebase CLI:n avulla

Mikä on Google Firebase CLI?

Google Firebase CLI (Command Line Interface) on työkalu, jonka avulla kehittäjät voivat hallita ja julkaista projektejaan Firebase Hosting -alustalla suoraan komentoriviltä. Firebase Hosting tarjoaa nopean ja turvallisen tavan isännöidä staattisia verkkosivustoja ja dynaamisia web-sovelluksia.

Lue lisää Firebase Hosting -palvelusta.

Esivaatimukset

Ennen kuin sukellamme hosting-prosessiin, varmista, että sinulla on seuraavat esivaatimukset:

  • Google-tili: Rekisteröidy Google-tilille, jos sinulla ei vielä ole sellaista.
  • Node.js: Varmista, että Node.js on asennettuna järjestelmääsi.
  • Firebase CLI: Asenna Firebase CLI globaalisti käyttämällä npm-komentoa kirjoittamalla terminaaliin:
    ‍
    npm install -g firebase-tools

Vaihe 1: Initialisoi Firebase projektissasi

Ensiksi, siirry projektihakemistoosi terminaalin avulla. Suorita sitten seuraava komento initialisoidaksesi Firebase projektissasi:

firebase login

Tämä komento kehottaa sinua kirjautumaan Google-tililläsi. Kun olet kirjautunut sisään, voit jatkaa Firebase

initialisointia projektissasi:

firebase init hosting

Seuraa kehotteita valitaksesi Firebase Hosting ja valitse projektisi listalta.

Vaihe 2: Julkaise verkkosivustosi

Kun Firebase on initialisoitu projektissasi, on aika julkaista verkkosivustosi. Varmista, että kaikki staattiset tiedostosi (HTML, CSS, JavaScript) ovat valmiina projektihakemistossa. Suorita sitten seuraava komento julkaistaksesi verkkosivustosi:

firebase deploy

Firebase CLI lataa tiedostosi Firebase Hosting -palveluun, ja kun julkaisu on valmis, se antaa sinulle URL-osoitteen, jossa verkkosivustosi on nyt julkisesti saatavilla.

Vaihe 3: Oma verkkotunnus

Jos sinulla on oma verkkotunnus, voit helposti liittää sen Firebase Hosting -palveluun. Varmista ensin, että olet vahvistanut verkkotunnuksesi omistajuuden Google Search Consolessa. Seuraa sitten näitä ohjeita:

  1. Mene Firebase-konsoliin ja valitse projektisi.
  2. Siirry kohtaan "Hosting"
  3. Klikkaa "Add custom domain" ja seuraa ohjeita lisätäksesi verkkotunnuksesi.
  4. Päivitä DNS-tietueesi annetuilla arvoilla. Mitä ovat DNS-tietueet?

Yhteenveto

Verkkosivustosi isännöiminen Google Firebase CLI avulla on yksinkertainen prosessi, joka tarjoaa nopeutta, turvallisuutta ja skaalautuvuutta. Muutamalla komennolla voit julkaista verkkosivustosi ja tavoittaa maailmanlaajuisen yleisön nopeasti.
